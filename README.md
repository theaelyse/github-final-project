## Simple Interest Calculator

This project is a simple interest calculator written in Bash.  
It helps users calculate the simple interest based on principal amount, rate of interest, and time.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Input Parameters

The script accepts the following inputs from the user:
- **Principal (p):** The initial amount of money
- **Rate (r):** Annual rate of interest (in percentage)
- **Time (t):** Time period in years

## How to Run the Script

1. Open a terminal
2. Run the script using:
   ```bash
